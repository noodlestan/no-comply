# @purrtrait

> Render codebase related docs and metadata.

Purrtrait provides rendering and visualization libraries for codebase documentation and metadata. Builds on Purrception's extraction layer to produce human-readable output.

## Packages

### Libraries

- [@purrtrait/code-renderer](./libs/code-renderer/README.md) — Code layout models.
- [@purrtrait/lang-ts](./libs/lang-ts/README.md) — @purrception/lang-ts support.
- [@purrtrait/client-tsx](./libs/client-tsx/README.md) — In-browser TS/TSX parsing.
- [@purrtrait/solid-code](./libs/solid-code/README.md) — SolidJS code renderers.
- [@purrtrait/view-tsx](./libs/view-tsx/README.md) — Editable TSX code models.
