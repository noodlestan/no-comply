## Layering

```
External Libraries (@no-comply/*, @purrception/*, @purrtrait/*, @purrpose/*)
       ↓
   src/providers/  ────────→  src/services/  ──────→  src/app/navigation/
       ↓                              ↓
   src/app/ (providers wrap screens, screens consume providers via hooks)
       ↓
   src/app/screens/  →  src/app/templates/  →  src/app/components/
       ↓
   src/app/content/  (types, factories, examples used by screens)
```
