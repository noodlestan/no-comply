## Main APIs

### Type Expression Node — `TypeExpressionNode`

The core discriminated union representing a TypeScript type expression. 18 kind variants covering references, primitives, objects, arrays, tuples, intersections, unions, functions, components, utility types (Pick, Omit), literals, template literals, operators, mapped types, conditionals, and inference.

### Declaration Types

Wraps parsed/declared symbols in typed containers (`TypeExpressionDeclaration`, `InterfaceDeclaration`, `FunctionDeclaration`, `ComponentDeclaration`) with attached JSDoc metadata.

### Resolve Engine

Context-based recursive resolver that traverses `TypeExpressionNode` trees, resolves references via `resolveTypeDeclaration`, flattens intersections/unions, handles utility types, and detects circular references.
