# UI System

> Component library and UI services built with [SolidJS](https://www.solidjs.com/).

## Development

### Getting started

Make sure you [README](https://github.com/noodlestan/ui-system/blob/README.md) first.

Launch the application in dev mode:

```
nvm use
npm install
npm run dev
```
