# Standard UI

> Themeable component library built with [Context UI](https://github.com/noodlestan/context-ui) and [SolidJS](https://www.solidjs.com/).

## Development

### Getting started

Make sure you [README](https://github.com/noodlestan/context-ui/blob/README.md) first.

Launch the application in dev mode:

```
nvm use
npm install
npm run dev
```
