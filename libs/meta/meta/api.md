## Main API

### `createNoComplyMetaService(entities: NoComplyEntityData[]): NoComplyMetaAPI`

Factory that indexes an array of entity data and returns a query API.

#### `NoComplyMetaAPI` Methods

| Method                    | Signature                                     | Description                                      |
| ------------------------- | --------------------------------------------- | ------------------------------------------------ |
| `getEntities`             | `() => NoComplyEntityData[]`                  | Returns a defensive copy of all indexed entities |
| `getEntityMaybe`          | `<T>(pkg, type, name) => T \| undefined`      | Safe lookup by package / type / name             |
| `getEntity`               | `<T>(pkg, type, name) => T`                   | Lookup, throws on miss                           |
| `getPackageNames`         | `() => string[]`                              | Returns all package names                        |
| `hasPackage`              | `(pkg) => boolean`                            | Checks package existence                         |
| `getPackageModuleNames`   | `(pkg) => string[]`                           | Returns module names for a package               |
| `packageHasModule`        | `(pkg, mod) => boolean`                       | Checks module existence                          |
| `getModuleMaybe`          | `(pkg, mod) => ModuleEntityData \| undefined` | Safe module lookup                               |
| `getModuleSubModuleNames` | `(pkg, mod) => string[]`                      | Returns sub-module names                         |
| `getModuleEntities`       | `(pkg, mod) => NoComplyEntityData[]`          | Returns all entities in a module                 |

## Entity Types

Seven concrete entity types extend `NoComplyEntityData`:

| Type                   | Discriminator        | Specific Fields                           |
| ---------------------- | -------------------- | ----------------------------------------- |
| `ComponentEntityData`  | `type: 'component'`  | `group`, `component`, `factory`           |
| `ContextEntityData`    | `type: 'context'`    | `factories: string[]`                     |
| `ControllerEntityData` | `type: 'controller'` | `factories: string[]`                     |
| `MixinEntityData`      | `type: 'mixin'`      | `factories: string[]`                     |

## Helpers

### Generic Entity Helpers

| Function             | Signature                                    | Description                                                            |
| -------------------- | -------------------------------------------- | ---------------------------------------------------------------------- |
| `getEntityFunctions` | `(entity, filter?) => FunctionDeclaration[]` | Extracts function declarations from an entity's declared symbols       |
| `getEntityTypes`     | `(entity, filter?) => TypeDeclaration[]`     | Extracts type/interface declarations from an entity's declared symbols |

Both accept an optional `Filter` — either an array of names (whitelist) or a predicate function.
