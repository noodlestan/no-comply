# Agent Personalities

This file lists all available agent personalities.

Instructions on how to update it can be found at the bottom of this file.

## agent-context-curator

Use this skill when creating, editing, or planning repository context files, agent instructions, skills, processes, or technical guidance.

## agent-pair-driver

Use this skill along with pair-programmer when the user is setting direction and the agent is doing most of the edits.

## agent-pair-navigator

Use this skill along with pair-programmer when the user is doing most of the edits and the agent is calling the next step, reviewing, cleaning, and verifying.

---

# How to update this file and agent platform Links

- Use `update-skills` skill to update the skills index file.
- Use `update-agents` skill to update agent platform files for .opencode and .codex.
