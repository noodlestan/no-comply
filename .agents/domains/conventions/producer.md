# Conventions Producer API

**Use Cases:** Writing final conventions, defining new conventions, and drafting convention proposals.

**Provides:**

- Definitions
- Processes

**Capabilities:**

- Define new conventions files and drafts.
- Write convention drafts from friction.
- Write final conventions from drafts.

## Mandatory Reading

READ `definitions/index.md` - Essential definitions for conventions files and drafts.

### Definitions

| Type | Name | Definition | Path | Status |
| ---- | ---- | ---------- | ---- | ------ |

| Definition | Conventions Draft | A proposal to add or modify existing conventions that surfaced from friction experienced during a task, review, or correction session. | definitions/index.md | ✅ |
