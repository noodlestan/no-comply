# Task Related Files

## Main Files

### Domain Producer API Files (#all)

These files are auto-generated. They declare definitions, processes and commands for all agents that **generate structured records** or other type of content on this domain.

There is only one producer file per domain and is located at `.agents/domains/{domain-name}/producer.md`.

Example: `.agents/domains/tasks/producer.md`.

### Domain Consumer API Files (#all)

These files are auto-generated. They declare definitions, processes and commands for all agents that **read content from** or perform simple operations on this domain.

There is only one curator file per domain and is located at `.agents/domains/{domain-name}/curator.md`.

Example: `.agents/domains/tasks/curator.md`.
