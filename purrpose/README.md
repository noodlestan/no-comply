# @purrpose

> Single-purpose utilities (e.g. in-browser compilation).

This namespace provides focused, single-purpose utility packages primarily for in-browser code compilation and documentation tooling.

## Packages

### Libraries

- [@purrpose/client-babel](./libs/client-babel/README.md) — In-browser JSX/TSX compiler.
- [@purrpose/client-babel-preset-solidjs](./libs/client-babel-preset-solidjs/README.md) — SolidJS preset for above.
- [@purrpose/solid-shiki-service](./libs/solid-shiki-service/README.md) — Shiki syntax highlighting service for SolidJS.
