# Artificials Tools

> Agent entry points.

Deterministic operations for agents to work with art and context files.

This package is part of the [@artificials](../../README.md) toolkit.

## Development

Make sure you read the [@artificials README](../../README.md) first.

### Build

This CLI is packaged for use in Node.js environments. The entry point is built using `artificials-build` from `tools/`.
