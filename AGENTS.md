- When resolving a file path that contains a `$WORD` placeholder:

**Procedure:**

1. Interpret any placeholder according to the following table:

| Placeholder   | Meaning                                | Example usage                                    |
| ------------- | -------------------------------------- | ------------------------------------------------ |
| `$PROJECT`    | session project scope                  | `$PROJECT/src/controllers`                       |
