# UI System Sandbox

> Sandbox application to develop, showcase and document UI System features.

## Development

### Getting started

Make sure you [README](../../../../README.md) first.

Launch the application in dev mode:

```
nvm use
npm install
npm run dev
```

### Stack

- [Starlight](https://starlight.astro.build/) / [Astro](https://docs.astro.build)

### Troubleshooting

#### `<item> is of type any`

Seeing warnings around `getCollection()` invocations?

Run `npm run sync` to force Astro to update collection typings.
